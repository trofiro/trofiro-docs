Trofiro is designed to get out of your way so you can focus on your work. This guide covers the keyboard shortcuts, bulk operations, and advanced search techniques that experienced users rely on every day to move faster and reduce repetitive clicking.

## Keyboard Shortcuts

Press **?** at any time to display the full shortcut reference overlay. The most commonly used shortcuts are listed below.

| Shortcut | Action |
|---|---|
| `Ctrl / Cmd + K` | Open command palette |
| `Ctrl / Cmd + N` | Create new resource |
| `Ctrl / Cmd + F` | Search resources |
| `Escape` | Close modal / cancel action |
| `Ctrl / Cmd + S` | Save changes |
| `?` | Show all shortcuts |

Shortcuts that reference `Ctrl / Cmd` use **Ctrl** on Windows and Linux and **Cmd** on macOS.

## Bulk Operations

When you need to act on more than one resource at a time, bulk operations save you from repeating the same action individually. To get started:

1. Navigate to **Dashboard > Resources**.
2. Check the **checkbox** to the left of each resource you want to include. You can also check the header checkbox to select every resource on the current page.
3. A bulk-action toolbar slides up from the bottom of the screen. Choose from the available actions:
   - **Archive** — move all selected resources to the archived state.
   - **Delete** — permanently remove all selected resources. You will be asked to confirm before any data is deleted.
   - **Tag** — apply one or more tags to every selected resource at once.
   - **Move to Workspace** — transfer selected resources into a different workspace.

Deselect all resources by clicking the header checkbox again or pressing **Escape**.

## Search and Filtering

The search bar in Trofiro supports both plain-text queries and structured filter syntax, giving you precise control over what appears in your resource list.

**Plain-text search** matches against resource names and descriptions. **Structured queries** let you combine multiple criteria in a single expression using `key:value` pairs. For example:

```text
tag:urgent status:active
```

This query returns only resources tagged `urgent` that are currently in the `active` status. You can extend queries with additional filters:

| Filter key | Example | Description |
|---|---|---|
| `tag` | `tag:urgent` | Match resources with this tag |
| `status` | `status:active` | Match resources with this status |
| `created` | `created:>2024-01-01` | Match resources created after a date |
| `updated` | `updated:<2024-06-01` | Match resources last updated before a date |
| `field` | `field.priority:high` | Match a custom field value |

Combine as many filter keys as you need in a single query. Results update in real time as you type.

<Tip>
  If you regularly use the same combination of filters, save it as a **Saved View**. Apply your filters, then click **Save View** in the filter sidebar and give it a name. Saved views appear in the left navigation so you can jump back to them with a single click.
</Tip>

## Command Palette

The command palette is the fastest way to navigate Trofiro without touching your mouse. Open it with **Ctrl / Cmd + K** from anywhere in the application.

Once the palette is open, you can:

- **Navigate** to any page or resource by typing its name.
- **Create** a new resource, workspace, or workflow by typing the action name (for example, `new resource`).
- **Run actions** on the resource you currently have open, such as archiving it or adding a tag.
- **Open settings** panels directly by name.

The palette displays a list of recent and suggested commands below the search input. Use the **arrow keys** to move through suggestions and press **Enter** to execute the highlighted item. Press **Escape** to dismiss the palette without taking any action.
